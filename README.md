# 🎯 Special Shot

**A fast-paced multiplayer FPS prototype built with Godot 4.6**

Fight against other players in a lightweight online arena featuring real-time ENet networking, team-based gameplay, and responsive first-person controls.

</div>

---

# ✨ Features

- 🎮 First-person shooter gameplay
- 🌐 Multiplayer using ENet
- 👥 Team-based matches
- 🔫 Shooting mechanics with projectiles
- 🎯 Responsive FPS controls
- ⏸ Pause menu
- 🎨 Stylized low-poly environment
- ⚡ Built with Godot 4.6

---

# 🛠 Built With

- 🎮 **Godot Engine 4.6**
- 🌐 ENet Multiplayer
- 📝 GDScript

---

# 🚀 Getting Started

## Requirements

- Godot Engine **4.6** or newer

## Run the project

```bash
git clone https://github.com/GazizVR/special-shot.git
```

Open the project in **Godot 4.6** and press **F5** to run.

---

# 🌐 Multiplayer

The game uses **ENet** networking.

You can:

- Host a server
- Connect to an existing server
- Play in real time with multiple players

---

# 🎮 Controls

| Action | Key |
|---------|-----|
| Move | WASD |
| Look Around | Mouse |
| Shoot | Left Mouse Button |
| Pause | Esc |
